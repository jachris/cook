# Installation

Cook is currently only available on GitHub. You need `Git` and `Python 3`.

## Unix

```bash
$ git clone https://github.com/jachris/cook 
$ cd cook 
$ ./setup.py develop
```

## Windows

```bash
$ git clone https://github.com/jachris/cook 
$ cd cook 
$ py setup.py develop
```
